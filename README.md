This repository showcases various projects that demonstrate my capability to build data-driven financial tools.

## Projects

### 1. Financial Data Pipeline
A robust data processing tool that cleans and standardizes financial datasets. Features include handling missing values, removing duplicates, and transforming raw financial data into analysis-ready formats.


### 2. Scenario Forecasting Engine
Reads financial assumptions from Excel and generates multiple forecast scenarios (conservative, base, optimistic). Useful for financial modeling, budget forecasting, and sensitivity analysis.

### 3. Market Data Scraper
Fetches and filters financial market data from Yahoo Finance. Demonstrates ability to work with external data sources and extract meaningful financial metrics.

## Getting Started

Each project has its own README with setup instructions. To run any project:

1. Clone this repo
2. Navigate to the project folder
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python src/main.py`
